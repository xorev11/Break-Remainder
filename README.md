# Break Reminder - это плагин, предназначенный для Visual Studio Code.

**Автор:** Воробьев Евгений.
**Группа:** M3116.

## Описание проекта.

**Break Reminder** - это расширение, которое поможет пользователям VS Code не забывать об отдыхе от учебы или работы.

## Основной функционал.

1. Интерфейс состоит из 5 кнопок
    - Выбрать время для перерыва.
    - Пропустить перерыв.
    - Отложить перерыв на некоторое время.
    - Отключить плагин в текущей сессии.
    - Настроить время рабочего времени.
2. Звуковое уведомление.
3. Краткая рекомендация о том, чем можно заняться во время отдыха
